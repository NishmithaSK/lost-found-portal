# Lost and Found Portal

A web-based portal facilitating item recovery through real-time reporting, structured search, and persistent database tracking. Built with Python, Flask, and SQLite.

---

## Features

- **Item Reporting:** Simple forms to submit details about lost or found belongings.
- **Search & Filter:** Easily browse and filter reported items to match lost entries with found items.
- **Data Persistence:** Structured backend storage for tracking item status, descriptions, and contact information.

---

## Tech Stack

- **Backend:** Python, Flask
- **Database:** SQLite
- **Frontend:** HTML, CSS, JavaScript

## Getting Started

1. **Clone the repository:**
   `git clone [https://github.com/NishmithaSK/lost-found-portal.git](https://github.com/NishmithaSK/lost-found-portal.git)`
   `cd lost-found-portal`

2. **Install dependencies:**
   `pip install flask`

3. **Run the application:**
   `python app.py`

4. Access the web app in your browser at `[http://127.0.0.1:5000](http://127.0.0.1:5000)`.
